# 说明

**将语法分为词法和句法。**

## 词法

**主要知识点：**
- 词分类
- 词性转换
- 词缀（和词性转换有交集）
- 数词
- 音便（和数词有较大联系）

## 句法

> [!TIP]
> 词如何组成句子，如何用有限的词组成无限的句子。

主要知识点
- 基础
  - 从句
  - 活用
  - 语序
- 助词
  - 格助词
  - 接续助词
  - 终助词
  - 副助词
- 助动词
- 补助动词


另外，还有：

## 连语

> [!TIP]
> 多个语法点结合，难以分类到具体某个当中

比如
```
晴れるかもしれない
```

有终助词，副助词，动词可能形

## 语用

> [!TIP]
> 部分语法需要结合具体使用场景才能完美理解。

- 敬语
- 词语辨析
